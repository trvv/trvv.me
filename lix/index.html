Lix is an English language spelling reform prioritizing phonemic coherence for the General American dialect. It fully preserves the existing English alphabet.

Lix iz an Inglix langwidj speling riyfuwrm prayuwritayzing fuwniymuk kuwhiyruns fuwr qiy Djenrul Umeyrikun dayulekt. It fiwliy priyzurvz qiy igzisting Inglix alfubet.

Despite having 26 characters, English has ~45 phonemes. This makes it a surprisingly poor alphabet in the strict sense, but permitting some basic clustering we can get away with everything we need to.

Dispayt having 26 keyrikturz, Inglix haz ~45 fuwniymz. Qis meyks it ey surprayzingliy puwr alfubet in qiy stxrikt sense, but purmiting sum beysik klusturing wiy kan git uwey wic evriycing wiy niyd tew.
